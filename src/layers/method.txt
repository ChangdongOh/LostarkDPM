CharacterLayer
- initialize_character_stat(character_stat)
    - 각종 스탯 초기화 -> 스탯별 초기화 메소드 호출

- initialize_combat_stat()
- get_combat_stat(target)

- initialize_stat()

- initialize_damage()

- initialize_crit()
- get_crit_rate()
- get_crit_damage()
- update_crit(crit_rate_amount, crit_damage_amount)

- update_attribute_with_func(attribute_name, update_func)
    - attribute_name에 해당하는 속성을 update_func 람다함수로 업데이트
    - attack_power, additional_damage, crit_damage 등 모두 적용 가능


EngravingLayer
- apply_static_engravings()
    - static한 각인의 효과들을 CharacterLayer attribute 업데이트에 사용
- apply_one_engraving(engraving_dict)
    - 하나의 각인 효과를 적용

EquipmentLayer 메소드 추가 필요

SkillTreeLayer
- parse_skill_tree()
    - json parser를 통해 skill configuration 파싱
- validate_skill_preset()
    - 파싱된 skill preset validation
- get_skill_preset_by_name()
    - skill 이름들로 구성된 리스트 리턴
- get_skill_preset_by_detail()
    - skill 상세내용으로 구성된 리스트 리턴
- update_all_skill_info()
    - 상위 레이어 attribute들과 skill 기본 정보, 트라이포드를 통해 스킬 데미지 계산
    - 스킬 쿨타임 계산
    - 버프내용을 포함하여 skill_info를 구성

JewelRuneLayer
- FROSTFIRE_JEWEL_LEVEL_DICT
- NEMESIS_JEWEL_LEVEL_DICT
    - 보석 레벨별 효과에 대한 dictionary
- validate_jewel_rune()
    - 보석 개수 확인 및 스킬트리와 비교

* 룬은 skill_preset에 포함시켜야야 할듯
